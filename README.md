
# TextUtils

A simple React application for performing basic text manipulation tasks.

## Features

* Convert text to Uppercase
* Convert text to Lowercase
* Remove Extra Spaces
* Copy Text to Clipboard
* Clear Text
* Word Count
* Character Count
* Estimated Reading Time
* Responsive UI

## Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3

## Installation

```bash
git clone <repository-url>
cd textutils
npm install
npm start
```

## Usage

1. Enter your text in the input box.
2. Choose the desired operation.
3. View the updated text and statistics instantly.

## Learning Purpose

This project was built to practice React concepts including:

* Components
* Props
* useState Hook
* Event Handling
* Conditional Rendering
